The chart's caption, which will render below the chart.

[ Highcharts caption reference](https://api.highcharts.com/highcharts/caption)

## Usage

```jsx
<HighchartsChart>
  <Caption align="center">
    This is center aligned caption.
  </Caption>
</HighchartsChart>
```
